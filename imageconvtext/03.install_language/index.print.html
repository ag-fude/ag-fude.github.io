Install Languages

Install Language Package

ImageConvText use the system language package to extract text, so must install target language on Windows 10 or Windows 11.

Caution
  • ImageConvText only support Windows 10 and Windows 11. Mac & Linux not supported.

  1. Windows -> Settings -> Time & language -> Language & region:

  2. Click Add a language:

  3. Search target language:

  4. Click Install:

  5. After installation finished, then you can use function Get Platform Language Tags to list all installed languages.
